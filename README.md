# inv-IMP_exp_molecule

Experimental datasets of molecules:
1. J_mols - 1-3JCH, 1-3JNH, 1JCC, HCS, CCS, NCS
2. 2D_mols - only 2D correlations, HCS, CCS

### J_mols
Claire Dickson and Grace Armitage's molecules, with J values extracted.

### 2D_mols
- run_1, run_2: farthest-point sampled molecules from Bristol cheminventories
- run_3: medium nSPS scorers; farthest-point sampled molecules from Bristol cheminventories for nSPS 15 < x < 30
- run_4: high nSPS scorers; farthest-point sampled molecules from Bristol cheminventories for nSPS 30 < x < 45
- run_5: very high nSPS scorers; farthest-point sampled molecules from Bristol cheminventories for nSPS 45 < x < 60
